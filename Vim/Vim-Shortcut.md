# Vim Short Cut

| press |            설명            |
| ----- | :------------------------: |
| i     |        insert mode         |
| v     |        visual mode         |
| o     | 아랫라인의 insert 시작한다 |
| esc   |      back normal mode      |
| q     |           끝내기           |
| wq    |       저장후 끝내기        |
| q!    |         강제 종료          |
| wq!   |     강제 저장 후 종료      |
